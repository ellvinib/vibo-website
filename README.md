# VIBO IT CONSULTING BV - Corporate Website

Professionele bedrijfswebsite voor VIBO IT CONSULTING BV.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Vercel (hosting)

## Kenmerken
- Meertalig (NL/FR/EN)
- SEO-geoptimaliseerd
- Responsive design
- Lead generation formulieren
- Blog/insights sectie
- GDPR-compliant

## Development
```bash
npm install
npm run dev
```

## Deployment
Automatisch via Vercel bij push naar `main`.
